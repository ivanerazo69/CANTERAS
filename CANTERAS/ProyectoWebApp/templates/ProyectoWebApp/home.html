Optimización Canteras Web

Canteras Web es el segundo modelo de software para la gestión de Fuentes de Extracción. Su principal diferencia respecto a Canteras Movil es la de realizar el tratamiento de datos para desarrollar procesos de aprendizaje automatico y así lograr optener la fuente de extracción que cumpla con las caracteristicas de un proyecto determinado en vias terciarías en el Departamento del Cauca

Procesos Canteras Web

Canteras Web realiza el procesamiento de datos y genera resultados a partir del entrenamiento de la base de datos previamente manipulada. Este software ha sido construido con un modelo de aprendizaje automatico, especificamente haciendo uso de redes neuronales y con esto lograr resultados más proximos a la realidad. Es importante mencionar que cómo todo software entrenado con algun modelo de aprendizaje automatico no está excento de posibles confusiones, sin embargo la idea de este proyecto es lograr una mejor respuesta ante cualquier circunstancia.

Artificial Intelligence in Civil Engineering Por: Ivan Felipe Erazo Ceron Canteras Web

Canteras web esta construido con modelos de aprendizaje automatico con la finalidad obtener información acerca de las fuentes de extracción de materiales y logre seleccionar la adecuada según sus necesidades. El procesamiento se realizo con fuentes en las zonas de: Norte, Sur, Oriente y Centro del Departamento del Cauca. Las fuentes de extracción es el resultado de la base de datos que ha recolectado la oficina de infraestructura de la gobernación del Cauca y a través del Centro de Desarrollo Tecnológico para la Innovación de la Infraestructura Víal de la Universidad del Cauca se ha logrado capturar la información para su realización. **Nota: Este software ha sido desarrollado como ejercicio académico.
